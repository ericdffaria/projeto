-- CODIGOS DE GIT --

Verificar conexo com a repo remoto!
Vamos começar" Novo Repo remoto, enviar os arquivos.

    -- VERIFICAR SEU GIT SE ESTA CONFIGURADO --

git config --global user.name "mesmo nome que esta na conta github"
git config --global user.email "mesmo e-mail que esta na conta github"
git remote add origin endereço da repo remoto

Começando fluxo de envio de uma versão para repo remote.

    <git init = "criar um novo repo".>
    <git status = "verificar os status dos arquivos".>
    <git add . = "adicionar todos arquivos de uma vez no repo local".>
    <git add nome do arquivo = "adicionar um arquivo separado na repo local".>
    <git commit nome do arquivo -m "msg" = comentar en um arquivo expecifico para enviar ao repo remoto.>
    <git commit -a -m "msg" = comentar o envio de todos arquivos para repo remoto.>
    <git push origin nome da branch = "enviar os arquivos para repo remoto.">
    --
Fim de envio de um arquivo para repo remote.
    --
Puxar fluxo de arquivos de um repo remoto.
    <git pull = "Puxar as alterações do repo remoto">
    --
Clone - Clonar um projeto da repo remote.
    <git Clone = Clonar repositorio>
    --
Fim do pull e clone = puxando e clonando repo remote.

        -- CRIANDO BRANCH --

    BRANCHES - São ramificação apartir de um codigo fonte.
    Branch é uma forma de que o git separa as versões dos projetos.
    
    git branch - verificar os branch
    git branch <nome da branch> - criar uma nova branch.

    -- DELETANDO BRANCH -- 

    git branch -d <nome da branch>
    git branch --delete <nome da branch>

    -- MUDANDO DE BRANCH --

    git checkout -b <nome da branch> - Cria uma branch e muda automatico para a branch nova.
    
    --  UNINDO AS BRANCHS --

    git marge <nome da branch> - faz a uniao de duas branch.
